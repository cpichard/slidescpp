A possible encrypt function

void encrypt(const char *src, char *dst) {
    int key = 6;
    int index = 0;
    while (src[index] != '\0') {
        dst[index] = (((src[index]-'a') + key) % 26) + 'a';
        index++;
    };
}
  • An int in the function converts char to int
  • Can we get rid of index ?

Using pointers to iterate

void encrypt(const char *src, char *dst) {
    int key = 6;
    while (*src != '\0') {
